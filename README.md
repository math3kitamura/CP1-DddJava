## Descrição
Projeto em Java que simula o funcionamento básico de uma autoescola, utilizando conceitos de Programação Orientada a Objetos.

## Funcionalidades
- Cadastro de alunos
- Agendamento e cancelamento de aulas
- Verificação para realização de exame
- Resultado de aprovação ou reprovação
- Exibição de resumo dos alunos

## Estrutura
- **Aluno**: armazena dados e controla aulas e exame
- **AutoEscola**: gerencia os alunos e mostra o resumo
- **Main**: executa o programa

## Tecnologias
- Java
- IntelliJ
- Git/GitHub

## Como executar
1. Abrir o projeto no IntelliJ
2. Executar a classe `Main`
